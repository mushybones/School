HTML Kunskapssida

Vad är HTML och vad används det för?

HTML står för HyperText Markup Language och används för att skapa strukturen och innehållet på webbsidor. Det är ett märkspråk som använder taggar för att definiera olika delar av en sida.

Vad är ett element? Ge exempel.

Definierar objekt i HTML-struktur. Webbläsare utgår från element för att tolka struktur. Kan vara tomma. Exempel på element är rubriker: h1>.../h1> eller paragraf eller bild (img)

Vad är ett attribut? Ge exempel

Anger ytterligare information om taggar. Id och klass är attribut.

Vilka är de viktigaste skillnaderna mellan HTML5 och HTML4?

HTML5 standard, annorlunda sätt av innehåll än HTML 4. Olika versioner av samma språk. HTML4 använder mycket mer div som man gav attributet id som säger att det är ett div.

Vad är XML och XHTML?

XML är mer flexibelt än HTML, möjlighet att själv definiera element och datastruktur, används ofta för dataöverföring, mer kontroll över dess struktur. XHTML är mer strikt än HTML Syntax måste vara korrekt för att kunna fungera, mer konsekvent hantering av element kan leda till mer lättförståeligt och bättre strukturerat kod.

Vad är DOM, och varför är den viktig?

DOM= Document Object Model, Representation över hemsidans struktur, Viktigt att klient ska strukturera hemsidan. DOM kan ändra textinnehåll/attribut/stil av element, eller ändra på hemsidan. Genereras utifrån din HTML-kod, representation av hierarki, användbart för att låta klient modda sida.
